# jest-template

# Next.js アプリケーション立ち上げ
Next.js アプリケーションのルートディレクトに移動する。<br>
```cd ~/jest__template__app```
<br>
下記コマンドでアプリケーションを立ち上げる。（確認のため開発モード）<br>
```npm run dev```
<br>
リクストを送信する。<br>
```http://localhost:3000```

# 内容
 - 1画面内操作
 - axios
 - LocalStorage への格納



# 主要コマンド
テストの開始<br>
```npm run test```
<br>
SnapShot の更新<br>
```npx jest --update-snapshot```
