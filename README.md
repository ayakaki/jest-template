# jest-template

# Next.js アプリケーション立ち上げ
Next.js アプリケーションのルートディレクトに移動する。<br>
```cd ~/jest__template__app```
<br>
下記コマンドでアプリケーションを立ち上げる。（確認のため開発モード）<br>
```npm run dev```
<br>
リクストを送信する。<br>
```http://localhost:3000```

# API モックサーバ立ち上げ方法
モックサーバアプリのルートディレクトリに移動する。<br>
```cd ~/jest__template__apimock```
<br>
下記コマンドでアプリケーションを立ち上げる。<br>
```instant-mock```
<br>
リクストを送信する。<br>
```http://localhost:8100```
